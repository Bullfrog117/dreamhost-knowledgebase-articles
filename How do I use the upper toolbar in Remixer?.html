The toolbar that appears at the top of every page within Remixer allows you to quickly access various design and setting features.

Using the toolbar to select a Remixer function

Click the following buttons to open various pages related to Remixer features:

  1. DreamHost logo – opens the main 'Select Theme' page.
  2. Pages – displays all of your previously saved pages.
  3. Content – opens a page where you can store various website assets such as images, videos, audio, and so on.
  4. Editor – opens the editing panel for the most recent page.
  5. Publish – publishes the completed page to your domain.

Pages

Displays all of your previously saved pages:

You can perform the following actions on this page:

  • View all of your previously saved pages.
  • Open the editor to edit a page (by clicking the Edit button on the selected page).
  • Publish a previously saved page (by clicking the Publish button on the selected page).
  • Add a new page (by clicking the Add new page button at the top).
  • Make setting changes (by clicking the gear icon on the selected page).

Using the gear icon

  • Click the gear icon to open the Page Settings page.
The Page Settings page opens:

You can perform the following actions on this page:

  • Basic Settings – create a page title and URL. You can delete the page by clicking the red Delete Page button.
  • Style – select a new style (theme and color).
  • Domain – select a domain to publish the page to (you can click a link to purchase a new domain).
  • Marketing – add SEO details and Google Analytics for your new website.

Be sure to click the Save settings button when you finish.

Content

Opens the Manage Content page where you can load various assets to your website:

Images

  • Upload files from your computer. You can upload files to Remixer by either clicking the box or dragging and dropping files into the box.
  • Import images from various social media sites.
  • Add royalty-free images.
  • Import images and text from a website by entering a URL.

Videos

  • Add direct links to video content using a URL.
  • Import video from various social media sites.
  • Search YouTube.
  • Search Vimeo.

Audio

  • Add direct links to video content using a URL.
  • Import audio from various social media sites.

Markdown

  • Create a Markdown file to use on your website. The button opens an editor where you can edit, preview, save, and delete a Markdown file.

Content Groups

  • View your previously saved images, videos, and audio groups.

Editor

Opens the editor page for the most recent page:

  • Edit the various page elements.
  • See the How do I edit a page? article for further details.

Publish

When you finish editing your page, click the Publish button.

An animated rocket appears showing you the upload progress of your website:

When your website publishes to your domain, an updated icon on the upper right appears:

  • Click the View! button to open the live website.
