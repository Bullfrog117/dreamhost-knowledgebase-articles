php-ffmpeg is a library used to convert video and audio files. Here are some details:

  • php-ffmpeg is free
  • php-ffmpeg installation is similar to installing a perl module
  • php-ffmpeg is used for many youtube clones
  • unlike other scripts, php-ffmpeg supports all video types

View more information at:

  • PHP-FFMpeg

Usage

For an overview of FFmpeg, view the man page at the following link.

  • ffmpeg

The following is a very basic example of how to convert an .avi file to an .flv file.

You can run this command on your web server after logging in via SSH:

[server]$ ffmpeg -i inputfile.avi outputfile.flv
