Mail Mirroring lets you set one domain to automatically match another domain's email addresses.

For example, if you set blog.example.com to mirror example.com, all the email addresses you have individually set up on example.com will automatically work for blog.example.com.

This feature can only be used if example.com is using DreamHost-hosted mail service; if example.com is using Anti-spam filtering, you must enable it for blog.example.com as well.

Creating a Mail Mirror

  1. Navigate to the (Panel > ‘Mail’ > ‘Custom MX’) page.
  2. Click the Edit button to the right of the domain you want to create a mirror for. This domain will probably not have any email addresses created. You're going to use this domain to Mirror the domain that already has email set up.
  3. On the next page, scroll down to the 'Mail Mirror' section.
  4. From the dropdown, select the domain you want to Mirror mail for.
  5. Click the button 'Mirror this domain's email addresses now!'. A success message displays notifying you that it will take a few hours for the DNS records to update online.
