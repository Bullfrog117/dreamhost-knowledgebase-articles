It is sometimes necessary to update the version of PHP on your website. This is due to several factors mentioned below along with troubleshooting techniques to test your new version of PHP. This article is continually updated whenever a PHP change is set to occur at DreamHost.

View the following link for a timeline of PHP supported version:

  • http://php.net/supported-versions.php

It's highly recommended that you upgrade to the latest version of PHP as DreamHost will soon EOL PHP version 5.5. Please visit the following pages to view information on how to update to the latest PHP version:

  • How do I change the PHP version of my site?

How does the PHP upgrade affect me?

PHP 5.2

End-Of-Lifed (EOL’d) by php.net. This version is no longer available at DreamHost.

PHP 5.3

End-Of-Lifed (EOL’d) by php.net. This version is no longer available at DreamHost.

PHP 5.4

End-Of-Lifed (EOL’d) by php.net. This version is no longer available at DreamHost.

PHP 5.5

Soon to be End-Of-Lifed (EOL’d) by php.net and removed from the panel.

PHP 5.6

Available in the panel. This is the version DreamHost recommends.

PHP 7

Available in the panel. 

Changes that affect compatibility with PHP 5.4 and above

  • Register globals, magic quotes and safe mode were removed.
  • The break / continue $var syntax was removed.
  • The ini option allow_call_time_pass_reference was removed.
  • The PHP default_charset is now "UTF-8" within the distributed php.ini files, but still defaults to "".
