It's extremely important to create daily backups of your website data. If your site was ever hacked, or an update to website software broke the site, a backup allows you to immediately restore the site to a previous and functional version.

DreamHost now offers 'Site Saver', a Cloud backup tool available within your panel. This allows your website data to be backed up on DreamHost's cloud servers so you always have a safe offsite backup of your data in case anything happens to your site.

Enabling 'Site Saver' in the panel

  1. Navigate to the (Panel > 'Cloud Services' > 'Site Saver') page.
  2. Click the Enable Site Saver button at the bottom. This takes a few moments to set up.
  3. Click Roll back my website to enable the service.
  4. This guides you through the steps to select and domain to enable the backup on.

Disable 'Site Saver'

  1. Navigate to the (Panel > 'Cloud Services' > 'Site Saver') page.
  2. To the right, click the 'Edit Backup Settings' link.
  3. Click Cancel Website Backup
