Piwigo is open source photo gallery software for the web. It contains many features that assists with exhibiting and organizing your photography. Photos can be added via the administrative panel or any FTP client. Photo upload capabilities have been extended to be compatible with desktop applications like digiKam, Shotwell, Lightroom, iPhoto, Aperture, and also iPhone/Android applications. Piwigo offers features like browse by dates where it can use the data stored in the digital photos to organize photos in a calendar view and photo tagging that will facilitate a visitors ability to browse and filter through your photo galleries. Customization is available with the installation of plugins. Features and themes can be installed to change the appearance of your gallery. Administrative tools allow you to process photos in batch mode, validate comments and user photos, add permalinks and manage remote servers. Best of all, Piwigo is open source and free to use.
