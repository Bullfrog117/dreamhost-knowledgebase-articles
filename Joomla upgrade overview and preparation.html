• As of Nov 3rd 2015, all Joomla sites older than version 2.5.28 will be permanently disabled by DreamHost due to security vulnerabilities.
  • NO EXTENSIONS will be granted from this date. You must upgrade if you wish to continue using Joomla.

The following describes how to upgrade to a stable version of Joomla.

If you are no longer using your Joomla! install

You do not need to contact tech support if:

  1. You no longer use your site
  2. Domain is already hosted elsewhere or expired
  3. You no longer use the domain for any reason

Feel free to delete your Joomla! install.

Before you begin

Backup your files

Before you attempt an upgrade or data migration, please backup your site first!

You can find more information about backing up your files in the following articles:

  • FTP
  • Backup MySQL

Here's a recommended backup plug-in from the Joomla! Extensions Directory:

  • Akeeba Backup Core
