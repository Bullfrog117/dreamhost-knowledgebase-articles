There are a variety of ways you can check to see if CloudFlare is working on a website.

Viewing headers

Since CloudFlare inserts header information into requests, you’ll need a way to check these headers to see if CloudFlare is enabled and working on your website.

When looking at these headers, you’re looking for the following specific CloudFlare header:

CF-Cache-Status: HIT

View the CloudFlare Railgun article for details on how to view the headers in different web browsers.
