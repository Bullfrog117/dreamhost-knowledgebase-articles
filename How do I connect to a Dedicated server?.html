Connecting to a Dedicated server

If your Dedicated Server's name is ds12345, you can connect to it before adding any domains to it by using the following hostname:

  • ds12345.dreamservers.com

Once your domain is added, you can use that for the hostname. But, you can continue to use the servername if you wish.
