The editing panel allows you to quickly edit a page using a sleek WYSIWYG editor.

How do I access the editing panel?

There are three ways you can access the editing page:

  • Click the Start Editing button for a selected template on the main Select Theme page.
  • Click the Editor button in the upper toolbar. This opens the editing page for the most recent template.
  • Click the Edit button for a previously saved template on the All Pages page, which opens when you click the Pages button in the upper toolbar.

Where do I begin?

Each template includes grids that feature various components that you can edit for your website including text, images, videos, and more.

To begin editing a grid, simply hover over it with your mouse to view the various elements that you can edit:

Using the upper editing toolbar buttons

The following buttons appear at the top of the editor:

  1. Undo – Reverts your last edit.
  2. Redo – Redoes your last edit.
  3. New – Opens the gallery box where you can select a new component to add.
  4. < – The side arrow opens the page structure that shows you all the components on your page.

Using the side arrow

When you click the side arrow on the upper right (<), a box appears that displays all of the components on your page:

  • Click an item to quickly open the editing panel for that component.
  • Click the upper right > arrow to collapse the box.

Editing a template

When you're ready to edit a grid, hover over and click to highlight it:

  1. Teardrop icon – Opens the editor for the specific component. You edit the text, image, video, or audio in a grid by first clicking this button.
  2. Remix – Opens the component gallery where you can "remix" the content by selecting a new component option.
  3. Options – Opens the Markup tab where you can view the styling code within the section.
  4. Component name – Shows the name of the component that is selected. In this example, the component name is "Grid 3".
  5. Delete – Deletes the entire grid.
  6. Selection tool (:::::: ::::::) – Highlights the selected grid so you can edit it.
  7. Add Text – Opens an editor where you can add additional text, such as a caption.
  8. Image – When you hover over an image, the Image tab appears over it. Click the image to edit it.

When you are ready to make an edit, either click the element or the teardrop icon adjacent to it.

Editing text

Click the teardrop icon adjacent to the text you wish to edit:

  • An editing ribbon appears that allows you to adjust the alignment, styling, link, size, font, and color for the selected text.

Editing an image

Click the teardrop icon adjacent to an image you wish to edit:

  • The current image displays in the left pane where you can make adjustments to it such as zooming and flipping it.
  • To replace the image with one from your collection, click an image in the right pane. You must first load your image files using the Content page.
  • When finished, click the apply button.

Editing video, audio, or other files

Video

Click the teardrop icon adjacent to a video clip you wish to edit:

  • Select a video clip that you've previously saved on the Content page, or
  • Search YouTube to import a video (may be subject to copyright).

Audio

Click the teardrop icon adjacent to an audio clip you wish to edit:

  • Type a URL for a Soundcloud audio page to import an audio clip.
  • You can also include audio clips that you've previously saved on the Content page.
