Adding a Shell user

View the Enabling Shell Access article for instructions on adding a normal Shell user to your Dedicated server.

The Shell user is able to run basic Shell commands after logging into the server via SSH.

Adding an admin user

The normal Shell user cannot run sudo/root commands on the server.

If you need to run sudo/root commands, you must create an admin user. This can be created on the (Panel > 'Dedicated' > 'Admin Users') page.
