The following instructions are only if you have a shared hosting plan in addition to a VPS or Dedicated server.

You must remove all users from the shared server in order to proceed.

  1. Navigate to the (Panel > ‘Billing & Account’ > ‘Manage Account’) page.
  2. Click the 'Edit' link under the 'Ends' column all the way to the right. Three options appear:
    • Auto-Renews
    • End Today
    • End After This Billing Period
    After you select to end your hosting plan, the following confirmation appears:
  3. Click Ok to proceed.
    If you did not remove your existing shared users from the shared server, the following message appears:
    • Make sure to remove them first. You can then proceed with canceling the hosting plan.
