A few configuration steps are needed to use an SSL certificate with WordPress.

Add an SSL certificate to the domain

Make sure your site is properly set up for HTTPS. You need to have an SSL certificate applied to your site and possibly a Unique IP address depending on your requirements for the site. See the Unique IP? article for more information.

Update the Home and Site URL in WordPress

  1. Log into WordPress.
  2. Navigate to Settings -> General page.
  3. On this page you can change the URLs. Make sure to save your changes.

Update your wp-config.php file

  1. Log into your webserver via SSH or SFTP.
  2. In your WordPress site directory is a file named wp-config.php. Open this file in a text editor.
  3. ​Add the following two lines above the line that​ says "Stop Editing Here”: 
    define('FORCE_SSL', true);
define('FORCE_SSL_ADMIN',true);

If this is an existing site, you must also change all internal URLs to use the new secure URL. You can do this in two ways:

  • Use a plugin.
  • Use WP CLI to run a search-replace command.

Force the URL to redirect to https

You can force your site to redirect from http to https in 2 ways.

  • Use a WordPress HTTPS plugin
  • Update rules in your site's .htaccess file

Once all steps above are complete, all requests to your website will redirect to https.
